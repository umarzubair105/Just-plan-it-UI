How system works
A company account can have multiple projects. Each project can have separate teams. A resource can play different role in different projects. Time can be allocated for each resource per project. Time estimation is done based on role for each Deliverable. After time estimation, deliverable can be picked to plan. System will plan it in first un planned Release having sufficient time for related resources. Once prioritized deliverables are picked. Unplanned Releases can be frozen. Planned release can be started once release start time comes. Dashboard shows all about release and how to keep going. Artifacts can be maintained for company, per project, per release and per deliverable.
